## Form的CSS样式设置方式

Form的样式应该在 *自定义的Form类中* 的 **WIDGET** 里设置。

例如：

``` python
class MyForm(forms.ModelForm):
    class Meta:
        model = MyModel
        widgets = {
            'myfield': forms.TextInput(attrs={'class': 'myfieldclass'}),
        }
```
