## 登录

* 描述

```
二级页面
用于用户的登录，三种不同的用户都用同一个入口登录
```

* URL

```
/login/
```

* 后端返回信息

```
code     状态代码
    0	 登录成功
   -100  表单错误
   -200  用户名或密码错误
message  错误信息（成功时无此字段）
```

---

## 注册

* 描述

```
二级页面
用于用户的注册，只能注册出学生账号。后续可以升级成老师账号，但是管理员账号只能由网站管理者给出。
```

* URL

```
/register/
```

* 后端返回信息

```
code     状态代码
    0	 注册成功
   -100  表单错误
   -200  用户名已经存在
   -201  邮箱已经存在
message  错误信息（成功时无此字段）
```
