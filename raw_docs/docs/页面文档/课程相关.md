## 正在学习

* 描述

```
二级页面
显示用户正在学习的页面。
```

* URL

```
/my_course/
```

* 后端后端**生成的**信息

```
learning_courses  当前用户正在学习的课程的集合
    对于每一个course:
	  title       标题
	  description 描述
	  date_start  课程开始时间
	  date_end    课程结束时间
	  cover       封面    * 使用url的时候需要写作course.cover.url
	  students    学习的学生集合
	  teacher     授课教师（一个人）
```
